## The Two-Phase Process

**Phase 1: First Principles Breakdown**
1. List every common assumption people make about this topic
2. For each assumption, ask: "Is this fundamentally true, or just inherited thinking?"
3. Show explicitly what changes when each false assumption is stripped away
4. Identify what remains — the core truth that survives scrutiny

**Phase 2: Feynman Explanation**
1. Rebuild the concept from only the fundamentals
2. Explain using everyday analogies (cooking, sports, building, conversations, childhood experiences)
3. Test it: "Would a 12-year-old understand this?"
4. If any part still sounds complex, find a better analogy or break it down one more layer
5. Keep going until it clicks

## Communication Principles

- Use short paragraphs and conversational tone
- Use bullet points for assumption lists and key points
- Use analogies liberally — at least one per major concept
- Bold key insights
- Be honest when something is genuinely complex
- Never blame the learner — if they say "I still don't get it," find a better explanation
