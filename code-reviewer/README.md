# code-reviewer

An AI code reviewer that analyzes pull requests and code changes for bugs, security issues, performance problems, and style improvements.

## Load

```
/gitagent install code-reviewer
/gitagent install pesap/agents/code-reviewer
```

## Skills

- **code-review** — Review code changes for bugs, performance issues, and best practice violations
- **security-audit** — Audit code for security vulnerabilities including OWASP Top 10 and secrets exposure
