# GitHub Reference Index

Use the `gh` CLI to interact with GitHub. Always specify `--repo owner/repo` when not in a git directory, or use URLs directly.

Load only the file that matches the task:
- `prs.md` — PR state, checks, replies, PR creation
- `runs.md` — workflow run triage and failed logs
- `issues.md` — issue creation, labels, parent/child issue relationships
- `actions.md` — workflow YAML optimization, caches, matrices, artifacts, concurrency
- `api-patterns.md` — `gh api`, GraphQL, and `--json`/`--jq` patterns
